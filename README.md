# DFC · 拉力曲线绘制工具

交互式拉力–拉距曲线绘制与蓄能计算。

- **横坐标**：拉距 (cm)，默认 0–100
- **纵坐标**：拉力 (Lb)，默认 0–100
- **蓄能**：曲线下面积，梯形积分，输出 J 与 ft·lb

## 开发调试

```bash
npm install
npm run dev          # 浏览器调试
npm run dev:electron # 桌面窗口调试
```

## 打包绿色版（给客户）

```bash
npm run pack
```

产物在 `D:\dev\DFC-release\`：

| 路径 | 说明 |
|------|------|
| `DFC拉力曲线-1.0.0-绿色版.zip` | **发给客户的绿色版**（免安装） |
| `win-unpacked\` | 解压后的完整程序目录（打包中间产物） |

**客户怎么用：** 解压 zip → 进入 `DFC拉力曲线` 文件夹 → 双击 `DFC拉力曲线.exe`（弓箭图标）。

**为什么不用单文件 portable exe？**  
单文件绿色版是自解压外壳，外壳图标经常和程序图标不一致，且强行改图标容易把文件改坏。文件夹/zip 版图标与程序一致，也更稳。

## 计算说明

储存能量 E = ∫ F dx（梯形积分）。  
单位换算：`1 Lb · cm ≈ 0.04448 J`。
