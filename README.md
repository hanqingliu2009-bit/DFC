# DFC · 拉力曲线绘制工具

交互式拉力–拉距曲线绘制与蓄能计算（桌面 / 浏览器）。

仓库：https://github.com/hanqingliu2009-bit/DFC

## 功能概览

### 曲线编辑
- 在图上**单击**添加测点，**拖拽**调整，**双击**删除
- 右侧表格可精确输入 / 修改拉距与拉力
- 支持 **CSV 导入 / 导出**

### 曲线显示与积分
- **折线**：测点直线相连，按折线做梯形积分
- **平滑样条**：自然三次样条拟合；橙色为样条，灰色虚线为折线对照
- 平滑模式下在样条上**加密采样**，显示与蓄能计算使用同一套采样点

### 蓄能与指标
- **储存能量 (J)**：曲线下面积 \(E = \int F\,dx\)
- **每磅蓄能 / 每千克蓄能**：总蓄能 ÷ 峰值拉力
- **蓄能系数**：曲线下面积 ÷（½ × 拉距跨度 × 当前拉力）  
  （参考直角三角形：底 = 拉距，高 = 当前拉力）
- 鼠标沿曲线滑动时，动态显示**当前位置、当前蓄能、蓄能系数**，并画出参考三角形

### 单位制
- 右上角切换 **公制 / 英制**
  - 公制：拉距 **cm**，拉力 **kg**，每千克蓄能 **J/kg**
  - 英制：拉距 **in**，拉力 **Lb**，每磅蓄能 **J/Lb**
- 内部统一按 **cm + Lb** 存储与计算，界面切换只改显示与分母单位
- 换算：`1 Lb·cm ≈ 0.04448 J`；`1 in = 2.54 cm`；`1 Lb ≈ 0.4536 kgf`

### 视图缩放
- **滚轮**：以鼠标位置为中心缩放
- **Alt + 拖拽** 或 **中键拖拽**：平移
- 工具栏：**放大 / 缩小 / 适应数据 / 复位**
- 也可手动修改坐标轴范围

## CSV 导入说明

两列：拉距、拉力。建议带表头并标明单位。

程序会识别中英文关键词，例如：

| 表头示例 | 处理 |
|---------|------|
| `拉距厘米,拉力磅` / `draw_cm,force_lb` | cm + Lb |
| `拉距英寸,拉力磅` / `draw_in,force_lb` | in→cm，Lb |
| `拉距厘米,拉力千克` | cm，kg→Lb |
| `拉距,拉力`（无单位） | 默认按 **cm + Lb** |
| 无表头纯数字 | 默认按 **cm + Lb** |

也识别：行程、力值、公分、吋、公斤，以及 `cm` / `in` / `lb` / `kg` 等。  
同时出现冲突单位时：厘米优先于英寸，磅优先于千克。

导出时按当前单位制写出表头（公制 `拉距_cm,拉力_kg`，英制 `拉距_in,拉力_lb`）。

示例数据：`sample-data/draw-force-curve.csv`

## 开发调试

```bash
npm install
npm run dev            # 浏览器：http://localhost:5173
npm run dev:electron   # Electron 桌面窗口
```

## 打包绿色版（给客户）

```bash
npm run pack
```

产物目录：`D:\dev\DFC-release\`

| 路径 | 说明 |
|------|------|
| `DFC拉力曲线-1.0.0-绿色版.zip` | **发给客户**（免安装） |
| `win-unpacked\` | 解压后的完整程序（打包中间产物） |

**客户用法：** 解压 zip → 打开 `DFC拉力曲线` 文件夹 → 双击 `DFC拉力曲线.exe`。

> 单文件 portable exe（`npm run pack:portable`）可用，但自解压外壳图标容易异常，推荐仍用 zip 绿色版。

## 技术栈

- React + TypeScript + Vite
- Electron（可选桌面打包）
- 蓄能：梯形积分；平滑模式基于自然三次样条加密采样
